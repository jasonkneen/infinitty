## FAQ

### Q: Can I use this in my commercial product?

**A:** It depends on how you use it:
- If it's a component in a larger product: **Yes** (under AGPL-3.0)
- If it's the primary offering: **No** (need Commercial License)
- If you want to avoid AGPL copyleft requirements: **Yes** (with Commercial License)

### Q: Do I need to pay if I open-source my changes?

**A:** No. You can use the software under AGPL-3.0 for free if you:
- Share your modifications with the community
- Comply with AGPL copyleft requirements

### Q: What about the network copyleft (Section 13 of AGPL)?

**A:**
- Under AGPL-3.0: Yes, if you deploy modified software over a network, you must offer source code to users
- Under Commercial License: You can opt out of this requirement

### Q: Can I modify the software for commercial use?

**A:**
- Under AGPL-3.0: Only if changes are shared (copyleft)
- Under Commercial License: Yes, modifications can be proprietary
